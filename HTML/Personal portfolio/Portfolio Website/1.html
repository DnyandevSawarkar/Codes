1. AUTOMATIC GARAGE SYSTEM

Our system uses OpenCV and OCR (Optical Character Recognition) to read
the number plates of approaching vehicles, and then uses pytesseract

authorized vehicles.

3. SMOKE DETECTION USING MQ-2 GAS SENSOR

When the MQ-2 gas sensor senses smoke in the surrounding and when
the value of the smoke goes beyond 150, the Arduino will be signaled
to buzz the buzzer.
